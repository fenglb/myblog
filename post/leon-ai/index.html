leon一个开源的个人助理，可以训练成自个的智能助理软件。

一直以来，我就想搭建自己的语音智能助理，但是有不喜欢商业的语音助理，比如Siri，OK Google，小度，小易等等，觉得控制权在开发商那边，数据也在他们那边，没有任何控制权和隐私权。

leon这个项目，可以拿来玩玩。

以下是我翻译leon的的词汇表.

词汇表

你也许不认识一些文档中的词汇;这篇文章将有助于您更好理解这些词汇已经它们和Leon的关系.

操作

操作作为切入口.

Attachment: packages/calendar/todolist.py example of the To-Do List module including actions such as create_list, complete_todo, etc.

API

API代表*应用程序编程接口*，它允许项目的不同节点之间进行通信。这些节点可以是服务器、库等。

Leon APIs 的例子: - HTTP API 服务器和客户端数据交换. - WebSocket API 服务器和客户端双向通讯通道. - Python library API 允许开发者自定义函数调用Leon的模块.

ASR

ASR或*自动语音识别*是利用计算机硬件和软件技术来识别和处理人类语音。

Leon使用这个模块来识别语音.

Attachments: - app/js/main.es6.js ASR web app. - server/src/core/asr.js ASR server.

Answers

Answers 是 Leon 的应答模块. Each package has its own set of answers with different translations. 每个包package都包含它们自个的 answers 模块, 这个模块具有不同翻译版本.

Attachment: packages/checker/data/answers/en.json example of the Checker package English answers.

Brain

Leon 的 Brain 是它的核心部分, 负责执行modules, 应答, 理解语句等等.

Attachment: server/src/core/brain.js Leon’s brain.

Classifier

分类器是一种模型, 一旦经过训练可以通过算法输出一个结果, 这个结果将用于做出决定.

Leon 使用它来存储expressions训练输出的结果.

Expressions

Expressions 是 Leon 用来训练理解能力的数据集合. 每一个package都拥有自个不同语言版本的训练集.

Attachment: packages/checker/data/expressions/en.json example of the Checker package English expressions.

Modules

Modules 是 Leon 的技巧; 由于它们 Leon 才能发挥它的功能. Modules 可以包含一个或者无限个actions.
Modules 是 Leon 的技巧; 由于它们 Leon 才能发挥它的功能. Modules 可以包含一个或者无限个actions.

NLU

NLU (自然语言理解) 帮助计算机理解人类语言.

Leon 利用它来加载最合适的方法classifier. > Attachment: server/src/core/nlu.js Leon’s NLU.

Packages

Leon 的 packages 包含一个或者无限个modules. 你可以认为 packages 就是 modules 的集合. 这是 answers 和 expressions保存的地方.

Attachment: packages/leon example of the Leon package including modules related to Leon himself.

::: 提醒 完整的 package 清单在这里. :::

STT

STT, 语音转文字, 把语言转换成文字.

Leon has multiple STT parsers; you can choose one (or several) to configure. Leon 有多个 STT 过滤器; 你通过configure可以选择其中一个或者数个.

Attachment: server/src/stt/stt.js Leon’s STT.

Synchronizer

synchronizer 可以同步内容到云端或者您的设备(Google Drive等). 主要有moduels配置.

可以根据您的偏好来在configure为每个 module 设置来支持这个功能.

Attachment: server/src/core/synchronizer.js Leon’s synchronizer.

TTS

TTS 或者 (文字转语音) 把文字转换为音频输出.

Leon 有多个 TTS synthesizers; 你可以在configure设置.

Attachment: server/src/tts/tts.js Leon’s TTS.
